1) first check the version of the python -->> (py --version)   download latest2) create folder (Django Project etc) and inside that folder will create virtual enviroment to install django   inside it using command -->> (py -m venv virtualEnv)3) then will activate bat file inside (Django Project\mysite-env\scripts\activate.bat)  then enter4) In visual studio in (command palette) will select (python: select interpreter) 5) then inside (Django project) folder will run command ==>> (python -m pip install django)   5) to check version of django inside (Django project folder) will run (python -m django --version).6) then run where virtual environment is activated (django-admin startproject demo_project .)    	-> the starproject command assumes by use of (.) at the end that the curent folder is your project folder	   and creates -->> manage.py (the django command-line administrative utility for the project)	   7) and a sub folder (demo_project) contains different files like __init__.py, asgi.py, settings.py etc	8) then create an empty development database -->> (python manage.py migrate) 9) first time when the server will run it will create a default database calle SQLite  10) after making sure that virtual enviroment is running start the django development server --> (python manage.py runserver)11) then ctrl+click the url. to open default browser.
